## 🛠️ How to Run the Energy Simulator & AI Insights

To run the simulation and see the dynamic Gemini API insights, follow these steps:

1. Clone this repository.
2. Install the required Google Generative AI SDK:
   `pip install google-generativeai`
3. Export your Gemini API Key to your environment variables:
   * **Mac/Linux:** `export GEMINI_API_KEY="your_api_key_here"`
   * **Windows (CMD):** `set GEMINI_API_KEY="your_api_key_here"`
   * **Windows (PowerShell):** `$env:GEMINI_API_KEY="your_api_key_here"`
4. Run the simulation:
   `python energy_efficiency_sim.py`
