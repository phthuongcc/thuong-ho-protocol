# Thuong Ho Protocol (THP)
> A purified, constant-size state-purge protocol for post-quantum financial infrastructure.

---

### Technical Foundations
- **State Compression:** 256-ary Verkle Trees (IPA-based).
- **Quantum Resilience:** Lattice-based Cryptography (ML-KEM/LWE-scheme).
- **Performance:** $O(1)$ verification complexity for mobile/low-resource nodes.
- **Consensus:** Proof-of-Contribution (Vần công) - Energy efficient.

### The Vision
THP is a **"Digital Commons of Truth"** designed to facilitate transparent capital flows for multilateral institutions (JICA, IFC, World Bank) while preserving citizen privacy through Zero-Knowledge distillation. It serves as the digital backbone for the 2035 vision of the Ho Chi Minh City International Financial Center.
